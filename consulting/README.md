## 면담 규칙
- 진로관련 면담은 레벨별 최대 2회 신청 가능
  - 단, 생활관련 면담은 필요 시 신청 가능 (제한 없음)
- 진로면담 시 의미있는 면담을 위하여 면담사전메일을 작성 후 면담일 전날까지 발송해야 함
  - 메일을 보내지 않을 경우 자동으로 면담이 취소됨
