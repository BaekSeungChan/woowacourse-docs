#### git 터미널 환경 세팅
* 윈도우즈 운영체제 사용자: [git for windows](https://gitforwindows.org/)를 설치
* 맥 운영체제 사용자
  * 터미널을 실행한 후 `git --version` 명령을 실행한다.
  * git이 설치되어 있지 않으면 git을 설치할 것인지 물어보는 화면세어 설치 버튼을 클릭해 설치하면 된다.
  * [맥에 Git 설치하기](http://bkcarrier.tistory.com/35) 문서 참고
